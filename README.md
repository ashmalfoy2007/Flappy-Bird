#Flappy Bird – Python + Pygame
##Overview

Flappy Bird is a fun and lightweight Python game built using Pygame.
Your goal is simple: control the bird, fly through the gaps between pipes, and score points without colliding.
The game features smooth physics, randomized obstacles, and an auto-restart system—all in a clean, minimal interface.

##Features
-Smooth gravity-based bird movement
-Random pipe generation for endless gameplay
-Score tracking displayed live on the screen
-Collision detection with pipes and boundaries
-Automatic restart after Game Over
-Beginner-friendly and easy-to-read Python code

##Technologies/Tools Used
-Python 3.x
-Pygame for game graphics and real-time loop
-Random module for pipe generation
-Sys for system-level event handling

##How to Get Started

1. Clone the repository:
git clone https://github.com/your-username/FlappyBird-Pygame.git
cd FlappyBird-Pygame
2. Install the required library:
pip install -r requirements.txt
3. Run the game:
python flappybird.py

##How to Play & Test the Game
-Press SPACE to make the bird jump.
-Avoid hitting pipes, the top, or the ground.
-Each time you pass a pipe pair, your score increases.
-When you collide, a Game Over message appears and the game restarts automatically.
-Just follow these simple steps, and you’re ready to enjoy a smooth and addictive Flappy Bird experience!

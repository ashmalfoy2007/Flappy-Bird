# **Flappy Bird – Python + Pygame**

## **Overview**

Flappy Bird is a fun and lightweight Python game built using **Pygame**.
Your goal is simple: control the bird, fly through the gaps between pipes, and score points without colliding.
The game features smooth physics, randomized obstacles, and an auto-restart system—all in a clean and minimal interface.

---

## **Features**

* Smooth gravity-based bird movement
* Random pipe generation for endless gameplay
* Score tracking displayed live on the screen
* Collision detection with pipes and boundaries
* Automatic restart after Game Over
* Beginner-friendly and easy-to-read Python code

---

## **Technologies / Tools Used**

* **Python 3.x**
* **Pygame** for game graphics and real-time game loop
* **Random** module for generating pipe heights
* **Sys** module for system-level event handling

---

## **How to Get Started**

### 1. **Clone the repository:**

```bash
git clone https://github.com/your-username/FlappyBird-Pygame.git
cd FlappyBird-Pygame
```

### 2. **Install dependencies:**

```bash
pip install -r requirements.txt
```

### 3. **Run the game:**

```bash
python flappybird.py
```

---

## **How to Play**

* Press **SPACE** to make the bird jump
* Avoid colliding with the pipes
* Avoid hitting the top and bottom boundaries
* Score increases by **+1** every time you pass a pipe pair
* Game restarts automatically after Game Over
